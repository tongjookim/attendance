# AI Style Attendance Pro (Intelligent Edition)

**AI Style Attendance Pro**는 워드프레스 사용자를 위해 설계된 지능적이고 현대적인 출석 체크 시스템입니다. 사용자 경험(UX)을 극대화한 AI 스타일의 세련된 대시보드와 인터랙티브한 플로팅 배너를 통해 커뮤니티 활성화를 돕습니다.

## 🌟 핵심 스마트 기능

* **AI 지능형 대시보드**: `[wp_attendance]` 숏코드를 통해 사용자의 연속 출석, 누적 기록, 현재 등급을 시각화된 데이터로 제공합니다.
* **인터랙티브 네온 배너**: 당일 미출석 사용자에게만 지능적으로 노출되는 무지개 네온 플로팅 배너가 출석 참여율을 자연스럽게 높여줍니다.
* **스마트 포인트 연동**: 기본 포인트와 연속 출석 보너스를 설정할 수 있으며, `myCred`와 연동되어 보상 체계를 자동화합니다.
* **엔터프라이즈급 데이터 관리**: 관리자 페이지에서 모든 출석 로그를 모니터링하고, JSON 기반의 백업 및 복구 기능을 통해 데이터를 안전하게 보호합니다.
* **최적화된 퍼포먼스**: AJAX 기술을 적용하여 페이지 새로고침 없이 빠른 출석 처리가 가능하며, 시스템 부하를 최소화했습니다.

## 🚀 빠른 시작 가이드

1.  **플러그인 설치**: `attendance.php` 파일을 워드프레스 플러그인 경로(`/wp-content/plugins/attendance/`)에 업로드합니다.
2.  **활성화**: 관리자 메뉴에서 **AI Style Attendance Pro**를 활성화하면 시스템이 자동으로 데이터베이스 테이블을 생성합니다.
3.  **포인트 설정**: **[출석 시스템] > [설정 및 데이터 관리]** 메뉴에서 운영 정책에 맞는 포인트를 설정합니다.
4.  **숏코드 배치**: 원하는 페이지에 `[sir_attendance]`를 삽입하여 AI 스타일의 출석 대시보드를 노출합니다.

## 🛠 기술 사양 및 보안

* **데이터 안정성**: 전용 DB 테이블(`{wpdb_prefix}attendance_logs`)을 통해 방대한 출석 데이터를 효율적으로 관리합니다.
* **보안 강화**: 모든 요청은 WordPress Nonce 보안 인증을 거쳐 비정상적인 접근을 차단합니다.
* **클린 코드**: 플러그인 삭제 시 생성된 데이터를 모두 정리하는 언인스톨 기능을 포함하여 사이트 무거워짐을 방지합니다.
